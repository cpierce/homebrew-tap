# Homebrew Tap

Custom Homebrew formulas by [Chris Pierce](https://github.com/cpierce).

## Installation

```bash
brew tap cpierce/tap
```

## Formulas

| Formula | Description |
|---|---|
| [l8nc](Formula/l8nc.rb) | Multi-target continuous ping monitor for network troubleshooting |

### l8nc

A CLI tool for monitoring latency across multiple targets simultaneously with real-time visualizations.

```bash
brew install cpierce/tap/l8nc
```

Usage:

```bash
l8nc google.com cloudflare.com 1.1.1.1
```

For more information, see the [l8nc repository](https://github.com/cpierce/l8nc).
